# Инструкция по использованию консольного приложения для фильтрации заказов

## Описание
Это консольное приложение предназначено для фильтрации заказов на доставку в зависимости от района и времени.

## Как запустить приложение

1. Убедитесь, что у Вас установлен .NET SDK.

2. Поместите файл `orders.csv` в ту же папку, что и исполняемый файл приложения.

3. Откройте терминал (Command Prompt или PowerShell) и перейдите в директорию проекта.

4. Используйте следующую команду для запуска приложения: dotnet run "<район>" "<время доставки>"

Пример:
dotnet run "District1" "2023-10-01 12:00:00"


## Формат ввода
- `<район>`: название района, например, "District1".
- `<время доставки>`: дата и время в формате `yyyy-MM-dd HH:mm:ss`.

## Логирование
Результаты выполнения и возможные ошибки записываются в файл `deliverylog.txt`.

## Замечания
- Приложение проверяет корректность введенных параметров. Если параметры заданы неверно, будет выведено сообщение об ошибке.
- Если в указанный временной период нет доступных заказов, приложение выведет соответствующее сообщение.
